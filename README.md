# Candy Gus

## Description
Candy Gus is a simple arcade-style game where you help **Gus** (my dog) collect treats while avoiding falling enemies.  
Move left and right, grab as many treats as you can, and try to survive as long as possible!

---

## Instructions to Play

- **← →**   Move Gus  
- **SPACE** Start the game  
- **ENTER** Restart after Game Over  

Goal:  
Collect treats to score points and avoid the enemies (cats).  
If you lose all your lives → **Game Over**.

The game is fully playable on any web browser.
